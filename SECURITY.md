# Security Policy

## Supported Versions

Only the latest version of the app is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < 5.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of this app seriously. If you have discovered a security vulnerability, please report it privately.

**DO NOT** open a public issue on GitHub.

Instead, please use the **Report a vulnerability** button on the **Security** tab of this repository, or visit:
https://github.com/darkrain-nl/home-assistant-addon-s0pcm-reader/security/advisories/new

### Disclosure Policy and Timelines

We follow a coordinated vulnerability disclosure process. 
- You can expect an initial acknowledgement of your vulnerability report within **7 days**.
- We aim to provide a fix or mitigation strategy within **30 days** of confirming the issue.
- Please allow up to **90 days** for a full coordinated public disclosure after a patch has been rolled out.
