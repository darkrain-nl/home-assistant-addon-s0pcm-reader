# Security Policy

## Supported Versions

Only the latest version of the addon is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < 3.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of this addon seriously. If you have discovered a security vulnerability, please report it privately.

**DO NOT** open a public issue on GitHub.

Instead, please use the **Report a vulnerability** button on the **Security** tab of this repository. We will strive to review and address the issue as quickly as possible.
